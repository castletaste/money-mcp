# Реализация MCP-сервера mcp-money

## Обзор

Построить полноценный MCP-сервер mcp-money по спецификации DESIGN.md: трекер расходов для AI-ассистентов с 16 MCP-инструментами, хранением в PostgreSQL через Drizzle ORM и stdio-транспортом. Публикуется как npm-пакет, запускается через `npx mcp-money`.

## Контекст

- Файлы: `index.ts` (переписать), новые файлы в `src/`
- Паттерны: Bun runtime, строгий TypeScript, `@modelcontextprotocol/sdk` для MCP
- Зависимости: `@modelcontextprotocol/sdk`, `drizzle-orm`, `postgres` (postgres.js), `drizzle-kit` (dev), `uuid` (v7)
- Примечание: DESIGN.md указывает postgres.js + Drizzle ORM (не Bun.sql) для совместимости с Node

## Подход к разработке

- **Подход к тестированию**: Обычный (сначала код, потом тесты)
- Полностью завершать каждую задачу перед переходом к следующей
- Использовать `bun test` для всех тестов
- Интеграционные тесты на реальной базе Postgres
- **КРИТИЧНО: каждая задача ДОЛЖНА включать новые/обновлённые тесты**
- **КРИТИЧНО: все тесты должны проходить перед переходом к следующей задаче**
